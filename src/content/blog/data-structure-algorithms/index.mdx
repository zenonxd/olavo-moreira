Este é o post inicial onde irei publicar meus estudos sobre Estrutura de Dados e Algorítmos.

Aqui, será possível entender o básico do assunto em pauta, vendo exemplos dos algorítmos mais comuns na ciência da
computação.

Também veremos como aplicar esses algoritmos e estruturas de dados em exercícios do Leet Code, focando em entrevistas
de emprego.

A ideia para acompanhar esse estudo, é ter ao menos alguma base de alguma linguagem e também conhecimento em orientação
a objetos, seja em: python, java, go e outras.

Também tenha uma certa base de como manipular String, Arrays, etc.
